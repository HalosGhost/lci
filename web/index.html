What is LCI?
LCI is an interpreter for the lambda calculus. It was developed by Kostas Hatzikokolakis as an assignment for the "Theory of Programming Languages" course in Univirsity of Athens. It supports many advanced features like recursion, operators and multiple evaluation strategies, all based on the pure calculus. It is FREE SOFTWARE licenced under the GNU General Public Licence (GPL).

More Info - Contribution
You can find more information about LCI in project's page at sourceforge.net. You can post any questions you may have at LCI's forums or mailing lists. There is also a mechanism to report bugs, send patches or suggest new features. If you like to help improving LCI by contributing code or joining the development team (in order to really become a "team" ;-) you are something more than welcome.
